<h1>Owl Game Engine</h1>
    The Owl game engine is being developed in C++ so I can learn more about Direct3D and its DirectX library. It is a simple 2D engine and single thread. I intend to add new features to the engine whenever possible and in the future I intend to make the engine multi-threaded support, D3D12 and if possible a game engine editor.
